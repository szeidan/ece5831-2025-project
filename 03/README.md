# Assignment 03 – LogicGate Implementation

This folder contains the implementation and tests for the **LogicGate** class using NumPy.

## Files
- **logic_gate.py** – Implements `LogicGate` class with AND, NAND, OR, NOR, XOR gates.
- **module3.py** – Test script that runs all gates and prints results.
- **module3.ipynb** – Jupyter notebook with explanations, comments, and examples.

## How to Run
Run a quick demo from the terminal:
```bash
python logic_gate.py --demo
